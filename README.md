#fancy

## Generate dist files

Install node [Node Installer](http://nodejs.org/download/)

```bash
$ npm install
$ gulp
```

## For dev

Generate css and scripts

```bash
$ cd fancy/
$ gulp sass
$ gulp scripts
```

or just

```bash
$ gulp
```

Start watch task:

```bash
$ cd fancy/
$ gulp watch
```

## For test

Run commands below:

```bash
$ cd fancy/
$ gulp
$ npm start
```

Then go to your browser and type the address [http://localhost:8000/test](http://localhost:8000/test)
