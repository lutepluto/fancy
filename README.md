#fancy
=====================

## How to run

1. install ruby [Ruby Installer](http://www.rubyinstaller.org/)

2. install sass:

```bash
$ gem install sass
```

3. install compass:

```bash
$ gem install compass
```

4. generate css:

```bash
$ cd fancy/ 
$ compass watch
```
or 
```bash
$ compass compile --productionss 
```
for production use
