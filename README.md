#fancy

## For dev

### Run gulp

Install node [Node Installer](http://nodejs.org/download/)

Run following command:

```bash
$ cd fancy/
$ npm install
```
Start watch task:

```bash
$ cd fancy/
$ gulp watch
```

Generate css:

```bash
$ cd fancy/
$ gulp sass
```
or just
```bash
$ gulp
```

## For test

Run commands below:

```bash
$ cd fancy/
$ npm start
```

Then go to your browser and type the address **http://localhost:8000/test**
